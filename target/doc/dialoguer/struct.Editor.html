Struct dialoguer::Editor

source · 
pub struct Editor { /* private fields */ }
Expand description

Launches the default editor to edit a string.

Example

use dialoguer::Editor;

if let Some(rv) = Editor::new().edit("Enter a commit message").unwrap() {
    println!("Your message:");
    println!("{}", rv);
} else {
    println!("Abort!");
}
